About:
  • Frontend: React (for building the user interface).
  • Backend: Node.js with Express (for creating the server and API).
  • Database: MongoDB (for storing game state, high scores, and user data).
  • User Security: Passwords are encrypted using bcrypt with a unique salt for each user.
  • Testing: Jest (for unit testing).
  • Other: HTML, CSS, JavaScript.
  • Score Calculation:
  • (wordLength * 100) + (isUnique ? 200 : 0) - (timeInSec * 2) - (attempts * 10)
  • Game Features:
  • Users can choose the word length, uniqueness, and language.
  • Players make guesses, with feedback provided for each guess.
  • High scores are calculated and stored in the database.
  • Players can spend coins to get hints.
  • Players are rewarded with coins for solving a word.
  • The number of coins depends on the word length:
    • 4 letters or less: 1 coin
    • 5–7 letters: 2 coins
    • 8–10 letters: 3 coins
  • Game sessions are automatically deleted after 60 minutes of inactivity.
  • Security: All game and user data is securely managed, and every request and validation is processed through the server to ensure full security.
