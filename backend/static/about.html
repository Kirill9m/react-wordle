About:
  • Frontend: React (for building the user interface).
  • Backend: Node.js with Express (for creating the server and API).
  • Database: MongoDB (for storing game state and high scores).
  • Testing: Jest (for unit testing).
  • Other: HTML, CSS, JavaScript.
  • The total score is calculated using the formula:
  • (wordLength * 100) + (isUnique ? 200 : 0) - (timeInSec * 2) - (attemps * 10)
